OBJECTIVES
---
A quizapp is a CRUD quiz web-app that an user can Create / Read / Update / Delete quiz questions and following answers. The results are shown at the end of quiz with user's answer / correct answers / scores. People may take the quiz again with the same quesitons or modify questions and take them.

User may login and see his/her result after finishing the quiz /
the answers for the questions are recorded so that an user can go back to the previous question to change his/her answers /
an user / administrator can add questions and select the answers for the corresponding questions in a separate page /

quizAPP was created as a homework for Database Design And Web Implementation. (CSCI-UA 0004-007, Spring 2016) It is deployed to school server (http://i6.cims.nyu.edu/~gl1144/hw7/index.php)
