OBJECTIVES
---
ReactBlogApp is a CRUD code blog web-app with Markdown Editor. I've wanted to contribute to the open-source community and share what I have studied and learned with code examples and images. 

I could share the projects on github but I found it hard to share more theoretical and development tips with people. Once I deploy this project, I help poeple to learn something from reading my articles

The frontend was coded in React and Redux for the state management, with scss for styling and responsive design. Also thie app uses Yarn to build and manage dependencies. Some of modules / libraries that I used are following
moment (for date formatting)
codemirror marked prismjs (for code linting and markdown formatting)

The admin can Create / Update / Delete Posts once he authenticates with password by clicking the login modal located in the footer. Of course, non-authenticated user can READ the post. The admin can make a post with images / markdown styling texts / blockquote / code linting (for css, javascript, html etc...)
